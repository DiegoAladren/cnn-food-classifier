# Red Neuronal Convolucional para clasificación de imágenes de comida

Este proyecto consiste en el desarrollo y entrenamiento de una red neuronal que detecta y clasifica imágenes de comidas obtenidas del siguiente dataset de kaggle: https://www.kaggle.com/datasets/trolukovich/food11-image-dataset

La parte relacionada con la creación del modelo se detalla en este notebook de Google Colab: https://colab.research.google.com/drive/1uwc5RUmsmYazoQ1JsSvxCn8TxIyvDJu9?usp=sharing

Posteriormente se desarrollará una aplicación con python que implemente este modelo hecho con TensorFlow para detectar imágenes de comida y clasificarlas.
